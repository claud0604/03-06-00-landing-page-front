# FaceFree Landing Page Frontend

Entry point and introduction page for the FaceFree beauty consultation platform.

## Part of FaceFree Platform
See the [main README](../../README.md) for full project documentation.

## Tech Stack
- Vanilla JavaScript, HTML, CSS
- No build step required

## Setup
```bash
git clone https://github.com/claud0604/03-06-00-landing-page-front.git
cd 03-06-00-landing-page-front
# Open index.html in browser or use a local server
```

## Deployment
Deployed on Cloudflare Pages. Auto-deploys on push to main.
